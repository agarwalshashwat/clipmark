# Ideas

This directory captures feature proposals, brainstorms, and exploratory thinking for the Clipmark project.

## Contents

| File | Description |
|---|---|
| `template.md` | Template for new idea documents |

## What belongs here

- Early-stage feature ideas that are not yet ready for a specification
- Brainstorming sessions and concept explorations
- UX / product direction discussions
- Research spikes ("what if we tried X?")

Ideas are **low-commitment** — they don't have to be technically complete or approved. They are a space to think out loud.

## Lifecycle of an idea

```
/ideas/<idea>.md  →  discuss & refine  →  /decisions/ADR-NNN.md  →  /specifications/<spec>.md
```

1. **Capture** — Write the idea in this directory using the template.
2. **Discuss** — Open a PR, gather feedback in review comments.
3. **Decide** — If the idea is adopted, create an ADR in `/decisions/`.
4. **Specify** — If the idea needs a full technical spec, add one in `/specifications/`.

## Naming convention

```
<YYYY-MM-DD>-<short-slug>.md
# examples:
2026-04-11-ai-auto-chapters.md
2026-05-01-offline-sync.md
```
